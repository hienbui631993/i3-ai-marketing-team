# Lead Magnet (PDF Guide) — i3 International

Produce a valuable, on-brand downloadable that earns the email opt-in. Design stays
simple so the **content value** leads (per the GUIDE's lead-magnet note).

## Before writing — load context
Read all of `marketing/context/`. Use brand colors (`#00588F`, `#002447`, accents)
and Helvetica Neue per `style-guide.md`; logo from `/i3 Media Kit/i3 Logo/`.

## Inputs to confirm
- Format (guide, checklist, buyer's guide, ROI worksheet, report) · topic & vertical
- The campaign/CTA it supports · target length (e.g. 8–12 pages).

## Method
1. **Promise** — one specific outcome the reader gets ("Cut retail shrink in 5 steps").
2. **Outline** — cover → intro → 3–6 value sections → summary/checklist → CTA page.
3. **Write** the content (Blog Writer voice rules apply: clear, structured, proof-backed,
   `[confirm]` on any stat).
4. **Produce the PDF** — extend the official **document/PDF skill**; apply i3 brand layer
   (colors, font, logo on cover + footer). Keep design clean and content-first.
5. **CTA page** — what to do next (demo/contact), tied to the campaign.

## Output
- Save source as `marketing/working/content/leadmagnet-<topic>.md` and the rendered
  `…-<topic>.pdf` alongside it.
- Note the landing-page/email hooks so those skills can promote it.

## Quality check
- [ ] One clear promise delivered; genuinely useful.
- [ ] On-brand cover, colors, font, logo; clean layout.
- [ ] Single CTA page; `[confirm]` for unverified claims.
