# Email / Newsletter — i3 International

Write emails that are useful, warm ("family"), and action-oriented.

## Before writing — load context
Read `marketing/context/brand-voice.md`, `product-offerings.md`, `marketing-strategy.md`.

## Inputs to confirm
- Email type (newsletter, nurture step, product/feature, event, announcement)
- Audience & funnel stage · the one CTA · source content (blog, campaign) if repurposing.

## Structure
1. **Subject lines** — provide 3 options (≤ ~50 chars), specific and non-spammy.
   Add 1 preview/preheader line.
2. **Opening** — warm, gets to the point; reader benefit in the first sentence.
3. **Body** — short, scannable; one core idea; bullets where helpful. Reference real
   i3 solutions/outcomes (no fabricated stats — `[confirm]`).
4. **One CTA** — a single clear button/link.
5. **Sign-off** — on-brand, human.
6. For a **sequence**, output each email with its goal, send-delay, and CTA.

## Voice
Affordable/trust-first, helpful, respectful of time. No hype, no filler openers.

## Output
- Save as `marketing/working/content/email-<topic>.md`.
- Include subject options, preheader, body, CTA, and (for sequences) the cadence plan.

## Quality check
- [ ] 3 subject options + preheader.
- [ ] One CTA; scannable; on-brand.
- [ ] Real solution names; `[confirm]` for any unverified claim.
