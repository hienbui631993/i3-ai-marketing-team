# Landing Page Builder — i3 International

Build a conversion-focused, on-brand landing page aligned to a campaign theme.

## Before building — load context
Read `marketing/context/style-guide.md` (colors, font, logo), `brand-voice.md`,
`product-offerings.md`, and the **Campaign Brief** if one exists (page must match its theme).

## Brand application
- Colors: Brand Blue `#00588F`, Dark `#0A1428` (hero/anchors); accents Pink
  `#FF007D`, Green `#00A661`, Yellow `#FFC107` (CTAs/highlights, sparingly).
- Font: **Montserrat** stack (`"Montserrat", Helvetica, Arial, sans-serif`).
- Logo: marketing logo w/ tagline (white version on the dark hero) from `/i3 Media Kit/`.

## Inputs to confirm
- Campaign/offer · target vertical & audience · primary CTA (demo, download, contact)
- Any required proof/sections · single page vs. with form.

## Required sections (in order)
1. **Hero** — headline (benefit/fearless hook), subhead, primary CTA button, supporting image.
2. **Problem / value** — the pain + i3's affordable-AI answer.
3. **Solution** — feature → benefit, naming real i3 solutions.
4. **Proof** — results/testimonials/logos (`[confirm]` any stat).
5. **How it works** — 3 simple steps.
6. **Secondary CTA / form**.
7. **Footer** — logo, contact, legal (© i3 International Inc. & i3 America Nevada Inc.).

## Build notes
- Output responsive, self-contained **HTML** (inline or `<style>` block); mobile-first.
- Scannable: short copy, clear hierarchy, one **primary** CTA repeated.
- Accessible: alt text, semantic headings, good contrast.

## Output
- Save as `marketing/working/pages/<campaign-slug>.html`.
- Note any images that need the Social Creative Designer (or Media Kit assets used).

## Quality check
- [ ] On-brand colors/font/logo; correct logo variant on hero.
- [ ] All sections present; one primary CTA; responsive.
- [ ] Matches campaign theme; `[confirm]` for unverified proof.
