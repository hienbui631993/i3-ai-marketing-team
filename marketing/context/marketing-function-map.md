## "Basic first" build order

Per your steer, we build the most-used, lowest-dependency skills first so the
team is usable quickly, then layer on the rest:

1. **Blog Writer** (Content Creator) — highest weekly volume, no external deps
2. **Social Post Copywriter** (Content Creator)
3. **Branded Deck** (Creative Designer) — reference-based, no MCP needed
4. **Campaign Brief / Strategy** (Campaign Strategist)
5. **Keyword Research** + **Market Research** (Market Researcher)
6. **Email / Newsletter** (Content Creator)
7. **Landing Page Builder** (Creative Designer)
8. **Campaign Report** + **Data Visualization** (Data Analyst)
9. **Lead Magnet** (Content Creator)
10. **Social Creative Designer** (Creative Designer) — **last**, needs an image-gen MCP set up

> Rationale: items 1–4 deliver value with zero external setup. The only skill
> that needs an MCP/API connection (image generation) is sequenced last so it
> never blocks the rest of the team.
