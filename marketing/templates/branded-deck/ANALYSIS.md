## Visual system
- **Dark, cinematic base** (`#0A1428`) with white type and **magenta (`#FF007D`) accents**
  for emphasis, dividers, and key words.
- **Full-bleed media** — video/imagery drives most slides; text sits over or beside it.
- **Montserrat** throughout (bold for headlines/section titles, regular for body).
- 16:9, generous negative space, one idea per slide.

## Narrative structure (the 18-slide flow to reuse)
1. **Cover** — logo / hero (full-bleed).
2. **About i3** — "37+ year award-winning, privately-held, family-driven" + the **4 key areas**.
3. **Section divider** — "i3 International's core applications".
4. **Safety & Security** — automated detection (access, facial similarity, slip-and-fall, loitering, LPR…).
5. **Asset Protection** — intelligent incident reporting (Smart-ER, walkouts, external theft, shrink…).
6. **Operational Insights** — loyalty, queue/overflow times, employee engagement, **Velocity drive-thru timer**, traffic.
7. **Predictive Analytics** — forecasting, fraud detection, multi-site dashboards, staffing.
8–15. **Vertical sections** — each = a **divider slide** (QSR / GROCERY / C-STORE / RETAIL) + a **value-prop slide**.
16. **Media / impact slide**.
17. **Partnership close** — "we treat our partners like family", values & long-term growth.
18. **Closing / contact**.

> This is a **corporate capabilities + verticals** deck. Reuse the flow for
> company/sales overviews; adapt sections for campaign- or product-specific decks.

## Slide-type patterns to replicate
- **Section dividers:** single large Montserrat word/phrase on `#0A1428`, magenta accent.
- **Capability slides:** short intro line + a clean bulleted capability list; supporting media.
- **Vertical value-prop slides:** 2–3 sentence positioning paragraph with key phrases
  bolded/magenta + product imagery.
- **Headlines:** sentence/lead-phrase style, bold Montserrat, white.

## How the skill should use this
1. Match the **template look** (Montserrat, `#0A1428`, `#FF007D`, white, full-bleed media, 16:9).
2. Reuse the **narrative flow** above for overview/sales decks.
3. Keep one idea per slide; bold/magenta the key phrases.
4. Use real i3 facts (capability lists above map to verified solutions in `product-offerings.md`).
5. Flag the last 10% (precise media placement, video embeds, margins) for manual polish.
6. **Don't bloat new decks** — the 327 MB comes from embedded video; only embed media when needed.
